"Titanic Data Science Project"

Project Overview:
This project is based on the Titanic dataset where I performed Exploratory Data Analysis (EDA) and built machine learning models to predict survival.

Files Included:
- Titanic.ipynb → Jupyter Notebook containing code and analysis
- Report.pdf → Detailed project report with explanations and visuals

Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

Key Steps:
- Data Cleaning
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Model Building
- Evaluation

Result:
Achieved model accuracy of approx 81%

How to Run:
Open the notebook in Jupyter Notebook or Google Colab and run all cells.
